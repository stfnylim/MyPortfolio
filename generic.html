Swaggy Shark

My 2D Side Scroller Game

What I did

I decided to try hashing out this game in order to expand my skills on both backend and frontend devlopment. I have gained skills such as just backend game development, such as player control, level creation, and user interface.

Obstacles

As I got more involved, I realized several obstacles that got in my way.

One of the ways I got stuck from lack of experience was connecting a database to an actual project. At the time, it was my first time attempting incorporating a database with C# code. As

Another way I struggled with the project was getting a bug with enemy generation. I had an enemy spawner that spawned a random enemy every increment of seconds. The game would work fine the first time I ran it, but once it replayed, the enemies would over spawn and the game was no longer possible to proceed. After many hours of research, I still couldn't figure out why the code wouldn't work, but I was able to try to critically think was wrong with my project. I tried to make sure I was checking both the distance and time for the enemy spawner and it finally was able to work properly again.
